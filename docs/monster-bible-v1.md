## 全100種 キャラクターデザイン設定書

**バージョン**: v1.0  
**世界観**: ナノスケールの生命エネルギー「ナノエナジー」が充満した地球で、  
人間の歩行エネルギーに引き寄せられて姿を現す微小生命体「ナノン」の図鑑。

---

## 命名規則

- **カタカナ2〜5音節**が基本。語感で性格が伝わること
- 語尾パターン：`〜ン`（丸い・可愛い）/ `〜ロ`（鈍重・岩系）/ `〜ス`（速い・鋭い）/ `〜ア`（神秘的）/ `〜キ`（元気・攻撃的）
- 進化後は語感が「重く・長く」なる傾向
- 同じ進化ラインは語根を共有する（例：チビフレア→フレアボム→インフェルノン）

### #001 ナノバナナ ✅実装済
**読み**: ナノバナナ  
**タイプ**: フォレスト / N  
**進化**: ナノバナナ → モリバナナン（Lv.20）  
**体長**: 15cm  
**生息地**: 草原・公園の茂み  

**外見**  
バナナが二本足で歩いているような姿。黄色い皮をマントのようにまとい、丸い黒目がのぞく。尻尾代わりに茎が生えており、走るとくるくる回る。

**性格**  
好奇心旺盛で人懐っこいが、驚くとバナナの皮を撒き散らして逃げる。転んだ相手を心配して近づいてくる天然な一面も。

**生態**  
朝露を舐めてエネルギーを補給する。群れで行動することが多く、列を作って移動する様子はとても愛らしい。

**スキル**  
- `リーフスラップ`: 葉っぱで素早く叩く基本攻撃
- `スリップ`: バナナの皮を投げつけ、相手のSPDを1ターン低下させる

**デザインメモ**  
バナナの黄色×草の緑を基調。目は水滴型で大きめ。足は小さな根っこ状。シンプルで子どもにも受け入れやすいデザインを意識。

### #002 コケモス ✅実装済
**読み**: コケモス  
**タイプ**: フォレスト / N  
**進化**: コケモス → グランモス（Lv.18）

**生態**

**スキル**  
- `スポア`: 毒胞子を撒き散らし、相手に毒状態を付与
- `かたいからだ`: 1ターンDEFを大幅アップ

**デザインメモ**  
苔の緑×土の茶色。ぶよぶよした質感。目は苔の隙間から覗く小さな赤い光点。フェルト生地のようなテクスチャーを意識。
