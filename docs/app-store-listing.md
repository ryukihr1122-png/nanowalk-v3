## 説明文（4000文字以内）

```
歩くだけでレアモンスターに出会える、新感覚コレクションゲーム。

📍 歩数がそのままゲームを動かす
毎日の歩数が冒険の燃料。歩けば歩くほどエンカウントチャンスが増え、レアリティの高いモンスターが姿を現す。ジムやランニング、通勤・通学がそのままプレイになる。

🌿 スカウト＆コレクション
出会ったモンスターはミニゲームでスカウト。タップタイミング・ルーレット・スライダーの3種のミニゲームを攻略して、モンスターを仲間に加えよう。スカウト成功率はストーン選択や特性ボーナスで変化する。

⬆️ 育成＆進化
同じモンスターをスカウトするたびにレベルアップ。一定レベルに達すると新たな特性が解放され、さらに進化も解禁。コレクターとしての達成感が止まらない。

🗺 エリア＆天候
現在地のエリアと天候がモンスターの出現に影響。公園、住宅地、川沿い…場所によって出会えるモンスターが変わる。季節や時間帯も出現率に関わってくる。

🎁 デイリー＆クエスト
毎日ログインするとストーンやアイテムがもらえる。クエストをこなしてNanoCoreを集め、限定モンスターを解放しよう。

▼ こんな人にぴったり
・毎日の歩数をもっと楽しくしたい
・コレクションゲームが好き
・ゆっくり育成を楽しみたい
・外出のモチベーションが欲しい

本アプリはApple HealthKitと連携し、歩数データを使用します。
位置情報の使用は任意です（使用しなくても遊べます）。
```
